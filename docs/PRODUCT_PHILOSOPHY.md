# Syntrix Product Philosophy

## Our Mission

Help multi-location businesses make better operational decisions.

Every screen exists to answer a question.

If a feature does not help answer that question, it should not exist.

---

# Daily Briefing

Purpose

Prepare me for today.

Questions Answered

• How did the company perform yesterday?
• What changed?
• What should I pay attention to today?
• Where should I spend my time?
• What schedule does the AI recommend?

This page is read once each morning.

It summarizes.

It does not investigate.

---

# Operations

Purpose

Show me where and what I need to investigate right now.

Questions Answered

• Is everything operating normally?
• Which locations are changing right now?
• Where are operational risks developing?
• Which issues require investigation?

This page is live.

It monitors.

It does not summarize yesterday.

---

# Locations

Purpose

Compare performance across every location, region, and district.

Questions Answered

• Which locations perform best?
• Which locations are declining?
• Which managers need support?
• Which regions outperform others?

This page compares.

It does not deeply analyze one location.

---

# Location Detail

Purpose

Understand and improve one location.

Questions Answered

• Why is this location performing this way?
• Which KPIs are causing problems?
• What trends exist?
• What recommendations does AI have?

This page investigates.

---

# AI Copilot

Purpose

Help me understand, decide, and act.

Questions Answered

• Why is this happening?
• What should I do?
• What is the expected impact?
• Can you help execute this action?

This page assists decision making.

---

# Product Principles

1. Show answers before data.

2. Every screen has one purpose.

3. Every recommendation explains why.

4. Every recommendation estimates business impact.

5. The user should never wonder what to do next.

6. Historical analysis belongs in Daily Briefing.

7. Real-time monitoring belongs in Operations.

8. Comparison belongs in Locations.

9. Investigation belongs in Location Detail.

10. AI helps users understand, decide, and act.
